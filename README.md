# 🚀 알고리즘 스터디 (NCT157)

다양한 언어(Java, JS, Python, C++)로 함께 성장하는 알고리즘 스터디 레포지토리입니다.

---

## 🐮 스터디 방식

### 📅 스터디 전
1. **개념 학습**: 매주 주차별로 지정된 알고리즘 개념을 순차적으로 학습합니다.
2. **문제 풀이**: 
   * **리뷰 문제**: 매주 공지되는 **필수** 문제 (반드시 풀이)
   * **과제 문제**: **선택** 사항 (일주일에 총 **4문제 이상** 풀이를 권장합니다.)
3. **코드 업로드**: 스터디 시간 전까지 본인의 폴더에 코드를 업로드합니다. (잔디 심기 및 기록용)

### 💬 스터디 당일
1. **풀이 공유**: 선정된 문제의 구조화 과정과 사용한 알고리즘 로직을 공유합니다.
2. **코드 리뷰**: 서로 다른 언어의 접근 방식을 비교하며 효율적인 로직을 탐구합니다.
